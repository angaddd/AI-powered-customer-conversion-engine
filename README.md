# AI-powered-customer-conversion-engine
Multi-tenant SaaS platform with a tracking script that captures user behavior, provides real-time dashboards, performs lead scoring (hot/warm/cold), and triggers AI-driven messaging and follow-ups.


This github repo contains backend code for AI-powered customer conversion engine

Tech stack used:
-django Framework
-Dokcer
-MYSQL database

Main architecture:
 -client website Sign up to multi-tenant saas website 
 -saas website provide them a tracking script which they use in their webiste. Tracking script tracks user behaviour and send to saas database.
 -Now application will provide score to your potential customer based on interactions. Filter them as hot, warm, and cold.
 - The AI model will send mail messages to your customer, To make conversion based on score.
 - In this way we can increase sales knowing intent of the customer.
